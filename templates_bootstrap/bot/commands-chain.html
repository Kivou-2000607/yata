 Module Chain  

This module provides tools to monitor chains and incoming attacks.

Command !chain  

Regularly gives status on the author's faction's chain (default = 15 minutes) and sends a notification to <@Role> (if given) when chain is about to timeout (default = 90s). It will stop watching on chain cooldown. Type !retal a second time will stop the chain watching.

Syntax: !chain or !chain or !chain <@Role> or !chain <@Role> timeout=<timeout (in seconds)> update=<update (in minutes)>

Command !retal  

Sends a message on incoming attacks of the author's faction. It mentions <@Role> if given. Several retals can be launched by different members from different factions. Type !retal a second time will stop the retal watching.

Syntax: !retal or !retal <@Role>

Scope: Anyone with AA permission can use it.

Command !fly  

Gives the list of members that are flying for faction <factionID> if given or author key holder's faction otherwise.

Syntax: !fly or !fly <factionID>

Command !hosp  

Gives the list of members that are hospitalized for faction <factionID> if given or author key holder's faction otherwise. Members are sorted by last action and direct link to profile is given.

Syntax: !hosp or !hosp <factionID>

Command !okay  

Gives the list of members that are okay for faction <factionID> if given or author key holder's faction otherwise. Members are sorted by last action and direct link to profile is given.

Syntax: !okay or !okay <factionID>

Aliases: !ok

Command !vault  

Gives the money and points faction vault balance of a user <tornId> if they belong to the author's faction.

Syntax: !vault <tornId>

Scope: Anyone with AA permission can use it anywhere.
