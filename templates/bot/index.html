The YATA bot is a discord bot made for Torn.
It has a set of appplication commands made to facilitate your Torn life while keeping your server secure.

Currently it has 2 modules (more might come ):

  • Verify
  • Notifications

Getting started with the setup

Before starting, know that you need to have admin permissions to fully setup the bot on your server. If so, follow these steps.
  1. Invite the bot following this link.
    By default it asks for administrator permission. This is not mandatory, however change them if you know what you are doing.
  2. Type / (and select the YATA bot if you have a lot of them) to check what it has in the belly. You might want to start with 
    /serverkeys add
  3. Try the different commands. If you encounter any difficulties, you can ask questions on our discord.

 Legacy bot

The Legacy YATA bot is a replica of the message commands bot that you might have used before the 30th of August 2022.
We had to stop delivering it for free on an unlimited number of servers for various reasons (more details on the forums).

The development of this bot is discontinued. Only major breakings that might occur because of API changes will be fixed.

If you still want to use it you'll have to pay 1 SE/month.
DM me on discord if you're interested.

Keep in mind that the long time goal is to implement its features to the free bot. However it's not likely to happen anytime soon.
