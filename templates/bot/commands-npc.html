  Module NPC  

This module handles general settings for NPCs.

Command /npc rangers  

  • dashboard the channel where an embed message will display NPC status.
  • alerts the channel where the alerts will be sent.
  • role the role to mention for the alerts.
  • timing the number of minutes between the alerts and the attacks.
NPC are currently attacked based on Loot Rangers team schedule. This command allows you to setup a dashboard channel where the status and the next attack are scheduled. You can also setup a different channel for alerts before the attacks.
