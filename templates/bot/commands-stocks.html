Module Stocks  

This module gives information on wssb and tcb sharing group and notifies good investments on the stock market.

Command !wssb  

Give education's time left and BB ownership for all @wssb.

Syntax: !wssb

Scope: @wssb can use it in #wssb.

NAME            | EDU TIME LEFT | WSSB
 --------------------------------------
userA           |  0d  5h 31min |  x
userB           |  5d 15h 56min |
userC           |  8d  8h 15min |
userD           |  8d 11h 55min |  x

{#

Use personal API keys to pull private information in a potentially public channel.

#}

Command !tcb  

Give bank investment's time left and BB ownership for all @tcb.

Syntax: !tcb

Scope: @tcb can use it in #tcb.

NAME            | INV TIME LEFT | TCB
--------------------------------------
userA           | 10d 13h 13min |
userB           | 14d 15h 54min |  x
userC           | 20d 16h 56min |
{#

Use personal API keys to pull private information in a potentially public channel.

#}

Command !trader  

Gives or removes the @Trader role of the command author.

Syntax: !trader

Scope: Anyone can use it anywhere.

{#

No key needed.

#}

Notifications  

Notifies @Trader role in #stocks for good investments on the stock market. Triggers are when:

  • forecast from bad to avg / good & shares available & price lower than trimestrial average
  • new shares are injected by the system

{#

No key needed.

#}
