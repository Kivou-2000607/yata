Module Admin  

Set of tools for administratin purposes.

Command !sync  

Sync the bot configuration with the dashboard. It needs to be done whenever changes in the server or in the dashboard has been made.

Syntax: !sync

Socpe: Only bot admins can do it.

Command !clear  

Delete the last unpinned messages of the channels in memory (about 100). Use !clear < n > to delete the last n messages.

Syntax: !sync

Socpe: You need manage messages permissions.

Command !suppress  

Removes the last embedded messages of the channel in memory (about 100). Use !suppress < n > to remove the last n embedded messages.

Syntax: !sync

Socpe: You need manage messages permissions.
