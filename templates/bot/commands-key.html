 Module user  

This module handles personal data such as API key storage for the bot.

{% comment %}

Note that key stored here will only be used by the player itself and not for passive features (like verification).

{% endcomment %}

Group /key  

A personal API key can be used for various features like starting war watch or getting notifications. The level needed depends on the features you use. Limited Access and faction API Access (AA) should cover all the cases.
  • /key add
    • key: the Torn API key you want to add.
    Enters your API key to the bot's database.
  • /key delete
    Deletes your API key from the bot's database.
Notes:
  • You can only enter a single key.
  • The key is available from all servers where the Yata bot is.
  • Admins don't have access to your key.
  • Your key will only be used from commands you issued yourself.
