Module Crimes  

This module provides tools to monitor organized crimes.

Command !oc  

Sends a message when an organized crime of the author's faction is ready (with all participants Okay). It mentions <@Role> if given. Several ocs can be launched by different members from different factions. Type !oc a second time will stop watching.

Syntax: !oc or !oc <@Role>

Scope: Anyone with AA permission can use it.

Syntax: !ocready
