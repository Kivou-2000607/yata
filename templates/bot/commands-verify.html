 Module Verify  

This module brings a set of features and commands to verify a discord account with a torn account. Verifying means:

  • Giving a special verified role (it can change depending on the servers)
  • Potentially giving faction and group of factions roles
  • Changing the discord display name to the Torn name

Command !verify  

Links a discord account with a torn account. If no arguments given it verifies the author of the command. It will never remove any role.

Syntax: !verify or !verify <TornID> or !verify <@DiscordMember>

Command !tag  

Do the same as !verify and append a tag (4 letters max) at the end of the nickname.

Syntax: !tag or !verify <tag>

Note: This function must be enabled by the server admins (disabled by default) and only work on self.

Command !verifyAll  

Loops over all members of the server and apply !verify only to non verified members. If force is asked is applies !verify to all members.

Syntax: !verifyAll or !verifyAll force

Command !checkFactions  

Loops over all factions and mention if a member is not part part of it. If force is asked it also removes the faction role and apply !verify.

Syntax: !checkFactions or !checkFactions force

On member join  

The bot automatically applies !verify to new members of a server. It sends a welcome message and the result of the verification in the system channel.
