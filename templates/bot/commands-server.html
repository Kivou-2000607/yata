  Module server  

This module handles general settings for the server.

Group /server keys  

The /server keys group of commands manages the keys that the bot will use for some tasks where the members don't need their keys (like verifications).
  • /server keys add
    • key: the Torn API key you want to add
    Adds an API key (no necessarly yours) to the server. Public keys are enough for the tasks that will use these keys. The more keys you add, the less pressure it puts on each one.
    • Notes
    • It is mandatory to setup at least one valid API key.
    • The bot will eventually leave your server if no valid keys are found.
    • A key of an inactive player (2 weeks) becomes invalid.
  • /server keys check
    Displays all valid keys and removes invalid ones. A dropdown Delete keys attached to the message will allow you to delete some keys.
